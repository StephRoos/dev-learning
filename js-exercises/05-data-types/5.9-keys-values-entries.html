5.9 — Object.keys, values, entries

Itérer sur les propriétés d'un objet

let user = { name: "Jean", age: 30, ville: "Paris" };

Object.keys(user)    // ["name", "age", "ville"]  — tableau des clés
Object.values(user)  // ["Jean", 30, "Paris"]      — tableau des valeurs
Object.entries(user) // [["name","Jean"], ["age",30], ["ville","Paris"]] — paires

// Itérer avec for..of
for (let key of Object.keys(user)) { }
for (let value of Object.values(user)) { }
for (let [key, value] of Object.entries(user)) { }

Transformations d'objets

Les objets n'ont pas de méthodes comme map ou filter. Pour les utiliser :

// Schéma : Objet → entries → transformation → objet
let salaries = { Jean: 100, Marie: 200, Pierre: 150 };

// Doubler tous les salaires
let doubled = Object.fromEntries(
    Object.entries(salaries).map(([name, salary]) => [name, salary * 2])
);
// { Jean: 200, Marie: 400, Pierre: 300 }

Fondamental : le pattern Object.fromEntries(Object.entries(obj).map(...)) est la façon standard de transformer un objet.

Bonus 1 — Inverser les clés et valeurs

Écris invertObject(obj) qui retourne un nouvel objet avec clés et valeurs échangées.

Bonus 2 — Filtrer un objet par ses valeurs

Écris filterObject(obj, predicate) qui retourne un nouvel objet avec seulement les entrées où predicate(value) est vrai.

Bonus 3 — Grouper un tableau par propriété

Écris groupBy(arr, key) qui groupe les objets d'un tableau par la valeur d'une propriété.

Note : Object.groupBy() existe nativement en JavaScript moderne (ES2024), mais l'implémenter à la main est un excellent exercice de reduce.
