5.7 — Map et Set

Map — Dictionnaire clé-valeur

let map = new Map();

// Contrairement aux objets, les clés peuvent être de n'importe quel type
map.set("string", 1);
map.set(42, "nombre comme clé");
map.set({}, "objet comme clé");   // possible avec Map !

map.get("string"); // 1
map.has("string"); // true
map.delete("string");
map.size;          // nombre d'entrées

// Itération (dans l'ordre d'insertion)
for (let [key, value] of map) { }
map.forEach((value, key) => { });

// Conversion objet ↔ Map
let map2 = new Map(Object.entries(obj));  // objet → Map
let obj2 = Object.fromEntries(map);       // Map → objet

Set — Ensemble de valeurs uniques

let set = new Set([1, 2, 3, 2, 1]); // déduplique automatiquement
// set = {1, 2, 3}

set.add(4);
set.has(2);     // true
set.delete(2);
set.size;       // 3

// Itération (dans l'ordre d'insertion)
for (let value of set) { }

Fondamental : Map > Object pour les dictionnaires dynamiques (clés de tout type, taille .size, ordre d'insertion). Set > Array pour tester l'appartenance (O(1) vs O(n)).
