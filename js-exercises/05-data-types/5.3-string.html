5.3 — Chaînes de caractères

Concepts clés

Backticks : permettent les templates littéraux `Bonjour ${nom}` et les strings multilignes.

Caractères spéciaux : \n (nouvelle ligne), \t (tabulation), \\ (backslash), \' (apostrophe)

Immutabilité : une string ne peut pas être modifiée, toute méthode retourne une nouvelle string.

let str = "hello";
str[0] = "H"; // silencieusement ignoré en mode normal
console.log(str); // toujours "hello"

// Accès aux caractères
str[0]       // "h"
str.at(-1)   // "o" (dernier caractère, avec .at() on peut utiliser des index négatifs)

// Recherche
str.indexOf("l")    // 2 (première occurrence)
str.lastIndexOf("l") // 3 (dernière occurrence)
str.includes("ell") // true

// Extraction
str.slice(1, 3)     // "el" (de l'index 1 à 3 exclu)
str.slice(-2)       // "lo" (les 2 derniers)
str.substring(1, 3) // "el" (similaire à slice, mais sans négatifs)

Fondamental : préférer slice à substring — plus cohérent et supporte les index négatifs.

Exercice 1 — Mettre la première lettre en majuscule

Écris une fonction ucFirst(str) qui retourne la string avec la première lettre en majuscule.

Pense au cas limite : que se passe-t-il si la string est vide ? str[0] retourne undefined dans ce cas.

Exercice 2 — Vérifier le spam

Écris une fonction checkSpam(str) qui retourne true si la string contient "viagra" ou "xxx" (insensible à la casse).

Convertir en lowercase avec toLowerCase() avant de chercher — c'est la façon standard de faire une comparaison insensible à la casse.

Exercice 3 — Tronquer un texte

Écris truncate(str, maxlength) : si la string dépasse maxlength, couper et ajouter "…" (un seul caractère unicode, pas trois points).

Attention : le "…" compte pour 1 caractère dans la longueur finale. Donc si maxlength = 20, tu retournes 19 caractères + "…".

Exercice 4 — Extraire le montant

Écris extractCurrencyValue(str) qui extrait le nombre d'une string comme "$120" et retourne un nombre.

Utilise slice(1) pour tout prendre à partir de l'index 1, puis convertis en nombre avec + ou Number().

Bonus 1 — camelCase converter

Implémente camelCase("my-short-string") → "myShortString"

Bonus 2 — Compter les occurrences

Écris countOccurrences(str, sub) qui compte combien de fois sub apparaît dans str.

Bonus 3 — Vérificateur de palindrome

Écris isPalindrome(str) qui retourne true si la string est un palindrome (ignore la casse et les espaces).

Pour inverser une string : str.split("").reverse().join("") — pattern classique à mémoriser.
