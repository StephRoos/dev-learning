5.4 — Tableaux

Concepts clés

Tableaux = objets spéciaux : les tableaux sont des objets avec des clés numériques ordonnées. Ils sont passés par référence.

// Déclaration
let arr = [1, 2, 3];
let arr2 = new Array(3); // tableau de 3 cases vides — rarement utile

// Modification (depuis la fin)
arr.push(4);   // ajoute en fin → [1, 2, 3, 4]
arr.pop();     // retire en fin → retourne 4

// Modification (depuis le début) — LENT sur grands tableaux
arr.unshift(0); // ajoute en début → [0, 1, 2, 3]
arr.shift();    // retire en début → retourne 0

// Itération
for (let item of arr) { }     // recommandé pour les valeurs
for (let i = 0; i < arr.length; i++) { } // si tu as besoin des index

Fondamental : ne jamais utiliser for...in sur un tableau — c'est fait pour les objets et peut parcourir des propriétés inattendues.
