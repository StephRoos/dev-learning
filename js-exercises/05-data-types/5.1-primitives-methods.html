5.1 — Méthodes des primitifs

Concept clé : Wrapper objects et autoboxing

Les primitifs (string, number, boolean) ne sont pas des objets. Pourtant on peut appeler des méthodes dessus.

Comment ? JavaScript crée temporairement un objet wrapper (String, Number, Boolean), appelle la méthode, puis détruit l'objet. Ce mécanisme s'appelle l'autoboxing.

let str = "hello";
// Quand tu écris str.toUpperCase(), JavaScript fait (en interne) :
// 1. new String("hello")  → objet temporaire
// 2. appelle .toUpperCase() sur cet objet
// 3. détruit l'objet → retourne le primitif "HELLO"

Fondamental : les primitifs restent légers, les objets wrappers sont temporaires.
