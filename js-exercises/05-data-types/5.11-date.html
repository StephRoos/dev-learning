5.11 — Date et heure

Créer des dates

new Date()                          // maintenant
new Date(timestamp)                 // depuis millisecondes epoch
new Date("2025-01-15")             // depuis string ISO 8601
new Date(year, month, day, h, m, s) // ATTENTION : month commence à 0 !

Getters principaux

let d = new Date();
d.getFullYear()  // 2025 (pas getYear !)
d.getMonth()     // 0-11 (0 = janvier)
d.getDate()      // 1-31 (jour du mois)
d.getDay()       // 0-6 (0 = dimanche)
d.getHours()     // 0-23
d.getMinutes()   // 0-59
d.getSeconds()   // 0-59
d.getTime()      // timestamp en millisecondes (depuis 1er janv 1970)

Autocorrection

// Les dates invalides sont autocorrigées !
let d = new Date(2024, 1, 32); // 32 février ?
// → autocorrigé en 3 mars 2024

// Très utile pour les calculs :
let nextMonth = new Date(2024, 0 + 1, 1); // 1er février

Performance

Date.now()         // timestamp actuel (plus rapide que new Date().getTime())
let start = Date.now();
// ... code à mesurer ...
let duration = Date.now() - start; // millisecondes écoulées

Fondamental : les mois commencent à 0 en JavaScript. C'est une source fréquente de bugs — toujours y penser.
