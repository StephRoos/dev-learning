5.5 — Méthodes de tableaux

La boîte à outils des tableaux

// MUTATION (modifient le tableau original)
arr.splice(index, deleteCount, ...items) // couper/insérer
arr.sort((a, b) => a - b)               // trier en place
arr.reverse()                            // inverser en place

// COPIE (retournent un nouveau tableau)
arr.slice(start, end)   // extraire une portion
arr.concat(other)       // fusionner deux tableaux
arr.map(fn)             // transformer chaque élément
arr.filter(fn)          // garder les éléments qui passent le test
arr.flat(depth)         // aplatir

// RECHERCHE
arr.find(fn)            // premier élément qui correspond
arr.findIndex(fn)       // son index
arr.indexOf(value)      // index d'une valeur exacte
arr.includes(value)     // boolean : est-ce présent ?

// ITÉRATION
arr.forEach(fn)         // exécuter une fonction sur chaque élément (retourne undefined)

// RÉDUCTION
arr.reduce((acc, curr) => ..., initialValue) // réduire à une seule valeur

// DIVERS
arr.join(separator)     // joindre en string
Array.isArray(value)    // vérifier si c'est un tableau

Fondamental : map, filter, reduce sont les trois piliers de la programmation fonctionnelle en JS.
