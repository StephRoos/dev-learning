2.11 - Opérateurs logiques

Concepts clés

  • || (OR) — retourne la première valeur truthy, ou la dernière valeur si toutes sont falsy
  • && (AND) — retourne la première valeur falsy, ou la dernière valeur si toutes sont truthy
  • ! (NOT) — inverse un booléen. !! convertit en booléen
  • Court-circuit : || s'arrête au premier truthy, && s'arrête au premier falsy
  • Valeurs falsy : false, 0, "", null, undefined, NaN
  • Priorité : ! > && > ||

Exercices

Ex1 — Résultats de OR

Prédis le résultat de chaque expression (sans l'exécuter d'abord) :

null || 2
null || 0 || "user"
null || 0 || undefined
null || 2 || "user"

|| retourne la première valeur truthy, sinon la dernière valeur.

Ex2 — OR avec des alerts (court-circuit)

Quel(s) alert(s) s'affichent ? Quelle valeur retourne l'expression ?

alert( alert(1) || 2 || alert(3) );

alert() retourne undefined. Le court-circuit s'arrête au premier truthy.

Prédiction : alerts affichés = ????, résultat final = ????

Ex3 — Résultats de AND

Prédis le résultat de chaque expression :

1 && 0
1 && 5
null && 5
0 && "no matter what"

&& retourne la première valeur falsy, sinon la dernière valeur.

Ex4 — AND avec des alerts

Quel(s) alert(s) s'affichent ?

alert( 1 && null && 2 );

1 est truthy → on continue. null est falsy → court-circuit, on retourne null.

Ex5 — Résultat de OR AND OR

Quel est le résultat ? (attention à la priorité : && avant ||)

null || (2 && 3) || 4

Évalue d'abord le groupe entre parenthèses, puis les ||.

Ex6 — Vérifier une tranche d'âge

Écrire une condition if qui vérifie si age est entre 14 et 90 (inclus).

Utilise && pour combiner deux comparaisons.

Ex7 — Vérifier hors d'une tranche

Écrire une condition qui vérifie si age est en dehors de la tranche 14-90.

Écris deux versions :

  • Version avec ||
  • Version avec ! (en niant la condition de l'Ex6)

Ex8 — Quel alert s'affiche ?

Prédis quels alerts s'affichent pour login = "Admin" :

let login = "Admin";

if (login == "Employee" || login == "Director" || login == "Admin" || login == "Marketing") {
    alert("Authorized");
} else if (login == "") {
    alert("Empty login");
} else {
    alert("Unknown user");
}

Bonus

B1 — Fonction canVote(age, nationality)

Écrire une fonction qui retourne true si une personne peut voter :

  • Doit avoir 18 ans ou plus
  • Doit être de nationalité "FR" ou "EU"
  • Ne doit pas être sous tutelle (underTutelage = false)

Teste avec plusieurs cas.

B2 — Valider un mot de passe

Écrire une fonction validatePassword(password) qui vérifie :

  • Au moins 8 caractères (password.length >= 8)
  • Contient au moins un chiffre (/\d/.test(password))
  • Contient au moins une majuscule (/[A-Z]/.test(password))

Retourner un message précis indiquant quelle règle est violée.

Utilise des && et || pour combiner les règles.

B3 — Simplifier des conditions imbriquées

Réécrire ce code en une seule expression logique :

function isEligible(user) {
    if (user.age >= 18) {
        if (user.hasAccount) {
            if (!user.isBanned) {
                return true;
            }
        }
    }
    return false;
}

Un if imbriqué est souvent un &&.
