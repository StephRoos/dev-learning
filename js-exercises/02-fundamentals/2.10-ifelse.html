2.10 - Opérateurs conditionnels : if, '?'

Concepts clés

  • if (condition) — exécute un bloc si la condition est vraie
  • else — exécute un bloc si la condition est fausse
  • else if — teste une nouvelle condition si la précédente est fausse
  • Conversion en booléen : valeurs "falsy" → 0, "", null, undefined, NaN, false
  • Opérateur ternaire : condition ? valeurSiVrai : valeurSiFaux
  • Le ternaire est une expression (retourne une valeur), pas une instruction

Ex3 — Afficher le signe d'un nombre

À partir d'une variable number, afficher :

  • 1 si le nombre est positif
  • -1 si le nombre est négatif
  • 0 si le nombre est zéro

Tu auras besoin de if / else if / else.

Ex4 — Réécrire if..else avec l'opérateur ternaire

Réécrire ce code en utilisant l'opérateur ? :

let result;
if (a + b < 4) {
    result = "Below";
} else {
    result = "Over";
}

Le ternaire s'écrit sur une seule ligne : let result = condition ? "A" : "B";

Ex5 — Réécrire if..else imbriqué avec ternaire

Réécrire ce code avec des opérateurs ternaires :

let message;
if (login == "Employee") {
    message = "Hello, Employee";
} else if (login == "Director") {
    message = "Hello, Director";
} else if (login == "") {
    message = "No login";
} else {
    message = "Unknown user";
}

On peut chaîner les ternaires : cond1 ? val1 : cond2 ? val2 : val3. Reste lisible !

B3 — Année bissextile

Une année est bissextile si :

  • Elle est divisible par 4
  • Sauf si elle est divisible par 100
  • Sauf si elle est divisible par 400 (dans ce cas elle EST bissextile)

Exemple : 2000 → bissextile, 1900 → non, 2024 → bissextile

L'opérateur modulo % donne le reste de la division.
