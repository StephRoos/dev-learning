2.14 - L'instruction "switch"

Concepts clés

  • switch (expression) compare l'expression avec chaque case via égalité stricte (===)
  • break est indispensable — sans lui, l'exécution "tombe" dans le case suivant (fall-through)
  • default s'exécute si aucun case ne correspond (optionnel, mais bonne pratique)
  • Groupement de cases : plusieurs cases sans break entre eux partagent le même code
  • Comparaison stricte : switch (x) avec x = "3" ne match PAS case 3: (nombre)
Piège classique : oublier un break provoque un "fall-through" — l'exécution continue dans le case suivant même si la valeur ne correspond pas !

Exercices

Ex1 — Réécrire "switch" en "if"

Convertir ce switch en chaîne de if/else if/else :

switch (browser) {
    case "Edge":
        alert("You've got the Edge!");
        break;
    case "Chrome":
    case "Firefox":
    case "Safari":
    case "Opera":
        alert("Okay we support these browsers too");
        break;
    default:
        alert("We hope that this page looks ok!");
}

Plusieurs cases groupés correspondent à un || dans le if.

Ex2 — Réécrire "if" en "switch"

Convertir ce code en switch :

let a = +prompt("a?", "");

if (a == 0) {
    alert(0);
}
if (a == 1) {
    alert(1);
}
if (a == 2 || a == 3) {
    alert("2,3");
}

Attention : utilise +prompt() pour avoir un nombre. Le switch compare en strict, donc le type doit correspondre.

Les conditions a == 2 et a == 3 peuvent être groupées avec deux cases sans break.

Bonus

B1 — Convertisseur jour (1-7) en nom français

Écrire une fonction getDayName(dayNumber) qui retourne le nom du jour :

getDayName(1) // "Lundi"
getDayName(7) // "Dimanche"
getDayName(0) // "Jour invalide"

B2 — Calculatrice simple avec switch

Écrire une fonction calculate(a, operator, b) :

calculate(10, "+", 5)  // 15
calculate(10, "-", 5)  // 5
calculate(10, "*", 5)  // 50
calculate(10, "/", 5)  // 2
calculate(10, "/", 0)  // "Division par zéro"
calculate(10, "^", 5)  // "Opérateur inconnu"

B3 — Classifier un caractère

Écrire une fonction classifyChar(char) qui détermine si un caractère est :

  • Une voyelle (a, e, i, o, u, y — majuscules incluses)
  • Un chiffre (0-9)
  • Un caractère spécial (tout le reste)

On peut utiliser switch avec des cases pour chaque voyelle/chiffre, ou combiner avec une logique hybride. Essaie les deux !
