3.1 — Débogage dans le navigateur

Concepts clés

  • DevTools : ouvre avec F12 ou Cmd+Option+I (Mac)
  • Panel Sources : voir et naviguer dans les fichiers JS chargés
  • Breakpoints : points d'arrêt cliquables dans la marge du code
  • debugger; : instruction qui déclenche un breakpoint depuis le code
  • console : console.log, console.table, console.group, console.error...
  • Watch : surveiller la valeur d'une expression en temps réel
  • Call Stack : liste des fonctions appelées pour arriver là où on est
  • Step over / into / out : avancer pas à pas dans le code

Exercice 3 — Maîtriser la console

Instructions

Utilise les différentes méthodes de console pour déboguer le tableau de produits :

  • console.log pour afficher une valeur simple
  • console.table pour afficher un tableau de façon lisible
  • console.group / console.groupEnd pour regrouper des logs
  • console.error pour signaler une erreur
  • console.warn pour un avertissement

Ouvre la console (F12 → Console) pour voir les résultats.
