4.4 — Méthodes d'objet et this

Concepts clés

  • Méthode : une fonction stockée comme propriété d'un objet
  • this : dans une méthode, this référence l'objet qui a appelé la méthode
  • this est dynamique : sa valeur dépend de comment la fonction est appelée, pas où elle est définie
  • Arrow functions : n'ont PAS leur propre this — elles héritent du this du scope parent
  • Perte de this : let fn = obj.method; fn() — this devient undefined (strict) ou global
// Syntaxe courte pour les méthodes (ES6)
let user = {
    name: "Alice",
    greet() {                        // équivalent à greet: function() {}
        return "Bonjour, " + this.name;
    }
};

// this dépend de l'appelant
let admin = { name: "Admin" };
admin.greet = user.greet;
admin.greet(); // "Bonjour, Admin" — this est admin

// Arrow function : this hérité, PAS redéfini
let timer = {
    name: "Timer",
    start() {
        setTimeout(() => {
            console.log(this.name); // "Timer" — this hérité de start()
        }, 100);
    }
};
