4.1 — Les Objets

Concepts clés

  • Création : let obj = {} (littéral) ou new Object()
  • Propriétés : paires clé/valeur, la clé est toujours une string (ou Symbol)
  • Dot notation : obj.name — pour les clés simples connues à l'avance
  • Bracket notation : obj["name"] — pour les clés dynamiques ou avec des espaces
  • Supprimer : delete obj.name
  • Tester l'existence : "name" in obj — retourne true/false
  • Boucle : for (let key in obj) — itère sur toutes les clés
  • Référence : les objets sont copiés par référence, pas par valeur
// Exemple complet
let user = {
    name: "Alice",
    age: 30,
    "is admin": true   // clé avec espace → bracket notation obligatoire
};

console.log(user.name);          // "Alice" — dot notation
console.log(user["is admin"]);   // true — bracket notation

let key = "age";
console.log(user[key]);          // 30 — clé dynamique via variable

delete user.age;
console.log("age" in user);      // false

for (let prop in user) {
    console.log(prop, ":", user[prop]);
}
