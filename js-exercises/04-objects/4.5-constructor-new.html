4.5 — Constructeur et opérateur new

Concepts clés

  • Fonction constructeur : convention PascalCase, appelée avec new
  • Ce que fait new :
    1. Crée un nouvel objet vide {}
    2. Assigne cet objet à this
    3. Exécute le corps de la fonction
    4. Retourne this implicitement
  • return dans un constructeur : si on retourne un objet → cet objet est retourné. Si on retourne un primitif (ou rien) → this est retourné.
  • Méthodes : on peut ajouter des méthodes dans le constructeur via this.method = function() {}
// Constructeur — PascalCase par convention
function User(name, age) {
    // new fait : this = {}
    this.name = name;
    this.age = age;
    this.isAdmin = false;

    this.greet = function() {
        return `Bonjour, je suis ${this.name}`;
    };
    // new retourne this implicitement
}

let alice = new User("Alice", 30);
let bob = new User("Bob", 25);

console.log(alice.name);    // "Alice"
console.log(alice.greet()); // "Bonjour, je suis Alice"
console.log(alice instanceof User); // true

B1 — Constructeur User avec validation

Crée un constructeur User qui valide les données en entrée. Si name est vide ou si age est négatif, lève une erreur avec throw new Error(...). 

// Ta réponse ici
function User(name, age, email) {
    // Validation
    if (!name || name.trim() === "") {
        throw new Error("Le nom est requis");
    }
    if (typeof age !== "number" || age < 0 || age > 150) {
        throw new Error("L'âge doit être un nombre entre 0 et 150");
    }
    if (email && !email.includes("@")) {
        throw new Error("Email invalide");
    }

    this.name = name.trim();
    this.age = age;
    this.email = email || null;

    this.toString = function() {
        return `User(${this.name}, ${this.age})`;
    };
}

// Tests

} catch (e) {
    console.error("Erreur capturée :", e.message);
}
