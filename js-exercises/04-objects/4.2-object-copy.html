4.2 — Copie d'objets et références

Concepts clés

  • Primitifs copiés par valeur : let b = a crée une copie indépendante
  • Objets copiés par référence : let b = a — b et a pointent vers le MÊME objet
  • Object.assign(dest, src) : copie superficielle des propriétés dans dest
  • Spread operator : let copy = { ...original } — copie superficielle aussi
  • Copie superficielle : les propriétés de premier niveau sont copiées, mais les objets imbriqués restent partagés
  • Copie profonde : structuredClone(obj) — copie récursive complète (ES2022)
// Référence partagée — DANGER
let user = { name: "Alice" };
let admin = user;     // même référence !
admin.name = "Bob";
console.log(user.name); // "Bob" — user est aussi modifié !

// Copie superficielle — OK pour objets plats
let copy = Object.assign({}, user);
// ou : let copy = { ...user };

// Copie profonde — pour objets imbriqués
let deep = structuredClone(user);
